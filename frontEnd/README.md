# Project Setup

## Setup Database Server

```sh
cd backEnd

```

```sh
python manage.py makemigrations

```

```sh
python manage.py migrate

```

### Run django server

```sh
python manage.py runserver

```

## Setup Frontend

```sh
cd frontend
```

```sh
npm install
```

### Run development server (will open at localhost:8080)

```sh
npm run dev
```
